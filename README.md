# Django Basic Form

Django Basic Form is a learning project which showcase how to create and maintain forms in Django.

## Features

1. Render Form.
2. Validate Form.
3. Submit Form Data.
4. View Submitted Data.


## How to Use this Project

1. GIT clone this Repo.
2. pip install -r requirements.txt
